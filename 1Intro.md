# Diseño e Integracion del Sistema DAQ Festo Labvolt con MATLAB Simulink y LabVIEW
**Autores:** Juan Diego Diaz Alvarez, Giuliana Andrea Caballero Guzman
   **Institución:** Universidad ECCI - Programa de Mecatrónica
    **Año:** 2026

Proyecto de investigación: Integración del DAQ Festo Labvolt con MATLAB/Simulink y LabVIEW para la validación de algoritmos de control en laboratorio.

## Descripción del Proyecto
Este proyecto busca solucionar la falta de comunicación fluida entre el hardware de adquisición de datos (DAQ) de Festo Labvolt y los entornos de desarrollo MATLAB/Simulink y LabVIEW. El objetivo es permitir que estudiantes e investigadores validen algoritmos de control (PID, ON/OFF) de manera práctica en el laboratorio académico.

## Objetivos
- **Objetivo General:** Diseñar e implementar una estrategia de comunicación entre el DAQ Festo Labvolt y los entornos MATLAB/Simulink y LabVIEW para validación de algoritmos de control.
- **Objetivos Específicos:**
  - Analizar las interfaces de comunicación (USB mediante DLL).
  - Integrar SDK con MATLAB (bibliotecas .NET).
  - Implementar comunicación con LabVIEW (VISA/DLL).
  - Validar algoritmos experimentales.

## Metodología
El proyecto sigue un enfoque experimental estructurado:
1. Revisión bibliográfica del SDK.
2. Configuración de controladores y librerías.
3. Desarrollo de bloques personalizados (S-Functions / VIs).
4. Pruebas de comunicación y validación experimental.

## Tecnologías Utilizadas
- **Hardware:** DAQ Festo Labvolt (licencia 9069-90).
- **Software:** MATLAB/Simulink, LabVIEW, .NET Framework CLR v4.0.
